Responsive Image

Responsive Image Components are used to display Images on websites, which are screen responsive. They expand upto the width of the container they are in.

Full Width Image

Full Width Image is the one that expands to cover the container it is present in.
[Resize the browser window, to see the responsive property.]

Round Image

Round Images are the circular, with a default dimension of 200px X 200px.

Square Image

Square Images are the square shaped, with a default dimension of 200px X 200px.
